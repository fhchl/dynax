Welcome to Dynax's documentation!
=================================

**Dynax** is a Python package for straight-forward simulation, fitting and
linearization of dynamical systems. It combines `JAX`_,
`Diffrax`_, `Equinox`_ and `SciPy`_ optimizers.

.. _JAX: https://github.com/google/jax
.. _Diffrax: https://github.com/patrick-kidger/diffrax
.. _Equinox: https://github.com/patrick-kidger/equinox
.. _Scipy: https://docs.scipy.org/doc/scipy/index.html

.. note::

   This project is under active development. The documentation is not there yet.
